# Cybersecurity AI Risk Analytics

## Industry
Cybersecurity

## Research Question
How can AI-assisted analytics help identify high-risk cybersecurity patterns and improve threat prioritization and incident response?

## Tools Planned
- Excel
- SQL
- Python
- Power BI
- AI-assisted analysis

## Project Goal
Analyze known exploited vulnerabilities, severity-related signals, remediation timelines, ransomware-linked entries, and vendor concentration patterns to identify insights that improve threat prioritization and security decision-making.

## Dataset
CISA KEV mirror dataset (`known_exploited_vulnerabilities.csv`), stored in `data/raw`.

## Suggested Dashboard KPIs
- Total KEV vulnerabilities
- Vulnerabilities added by month
- Top vendors by KEV count
- Ransomware-linked vulnerabilities
- Median remediation window (`dateAdded` to `dueDate`)

## AI Assistance Used
AI was used to support project planning, documentation drafting, analytical workflow design, SQL/Python starter logic, and dashboard framing. Final file structure, validation, and interpretation should be reviewed and adjusted manually.
