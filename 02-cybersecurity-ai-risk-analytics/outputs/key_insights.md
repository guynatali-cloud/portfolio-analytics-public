# Key Insights

- The KEV dataset currently contains **1545** known exploited vulnerabilities.
- The median remediation window between `dateAdded` and `dueDate` is **21 days**.
- Vendors with the highest number of KEV entries appear repeatedly, making vendor concentration a useful prioritization lens.
- Vulnerabilities tagged with known ransomware campaign use deserve a separate high-priority queue for remediation planning.
- This project can support Power BI dashboards for vendor risk, monthly KEV inflow, ransomware-linked issues, and remediation timelines.
