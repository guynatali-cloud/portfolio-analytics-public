# E-Commerce AI Customer Revenue Analytics

## Industry
E-Commerce / Retail

## Research Question
Does AI personalization increase long-term customer value — or only short-term conversions?

## Project Goal
Analyze e-commerce customer behavior to determine whether AI-driven personalization genuinely improves customer lifetime value (LTV), repeat purchases, and retention — or primarily boosts one-time conversions without lasting impact.

## Tools Used
- Excel
- SQL
- Python
- Tableau
- AI-assisted analysis

## Dataset
**Source:** E-Commerce Customer Behavior Dataset — Kaggle (free)
**Size:** 350 customers, 11 original features + 10 derived fields

## Repository Structure
- `data` — raw, cleaned, and sample datasets
- `excel` — workbook for KPI tracking and personalization analysis
- `sql` — schema and 7+ analytical queries
- `python` — cleaning, EDA, AI personalization scoring model
- `dashboard` — Tableau dashboard plan
- `ai-research` — prompts, research framing, AI disclosure
- `outputs` — KPI summaries, executive summary
- `screenshots` — charts for GitHub portfolio
- `notes` — workflow and upload instructions

## Key Findings
- **Gold members** have the highest average spend and LTV proxy
- **VIP tier** (AI-identified) shows significantly higher LTV than Low priority
- **Satisfaction correlates with spend** but not perfectly — some high spenders are unsatisfied
- **Discount usage** increases satisfaction but doesn't guarantee higher LTV
- **AI personalization scoring** effectively segments customers by long-term value potential

## AI Assistance Used
AI was used for repo structure, KPI framing, SQL/Python drafting, scoring model design, and documentation. Final analysis and publishing decisions were made by the analyst.
