# Operations AI Efficiency Analytics

## Industry
Operations / Supply Chain / Logistics

## Research Question
How can AI-assisted analytics improve workflow efficiency, bottleneck detection, and resource allocation in business operations?

## Project Goal
Analyze operational shipment data to identify delays, bottlenecks, carrier performance gaps, and throughput patterns — and evaluate whether AI-assisted efficiency scoring can improve decision-making for operations teams.

## Tools Used
- Excel
- SQL
- Python
- Power BI
- AI-assisted analysis

## Dataset
**Source:** Supply Chain Logistics Problem — Kaggle (free)
**Size:** 9,215 shipments, 14 original features + 9 derived fields

## Repository Structure
- `data` — raw, cleaned, and sample datasets
- `excel` — workbook for KPI tracking and carrier analysis
- `sql` — schema and 7+ analytical queries + AI bottleneck query
- `python` — cleaning, EDA, AI efficiency scoring model
- `dashboard` — Power BI dashboard plan
- `ai-research` — prompts, research framing, AI disclosure
- `outputs` — KPI summaries, executive summary
- `screenshots` — charts for GitHub portfolio
- `notes` — workflow and upload instructions

## Key Findings
- **Overall late rate:** 2.1% — most shipments on time or early
- **Delay severity:** 97.9% on time, with minor/moderate/severe tiers identified
- **Carrier analysis:** Different carriers show varying performance profiles
- **Service levels:** CRF, DTP, DTD each have distinct efficiency patterns
- **AI efficiency scoring:** Successfully separates high-performing from underperforming shipments

## AI Assistance Used
AI was used for repo structure, KPI framing, SQL/Python drafting, scoring model design, and documentation. Final analysis and publishing decisions were made by the analyst.
