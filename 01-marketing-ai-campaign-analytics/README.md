# Marketing AI Campaign Analytics

## Industry
Marketing

## Research Question
How can AI-assisted campaign analysis improve conversion performance and budget allocation across marketing channels?

## Project Goal
Analyze cross-channel campaign performance and use AI-assisted scoring to identify where budget should be reallocated to improve conversions, revenue efficiency, and ROAS.

## Tools Used
- Excel
- SQL
- Python
- Tableau
- AI-assisted analysis

## Repository Structure
- `data` - raw, cleaned, and sample datasets
- `excel` - workbook for KPI tracking and budget simulation
- `sql` - schema and analytical queries
- `python` - scripts for cleaning, EDA, and AI-assisted scoring
- `dashboard` - Tableau dashboard plan and publishing notes
- `ai-research` - prompts, research framing, and AI-use disclosure
- `outputs` - summary tables and recommendations
- `screenshots` - charts and process visuals for GitHub
- `notes` - workflow and upload instructions

## Key Questions Answered
1. Which platform delivers the strongest ROAS?
2. Which campaign type converts best?
3. Where is ad spend currently inefficient?
4. How should budget be reallocated across channels?
5. How can AI assistance accelerate analysis without replacing analyst judgment?
