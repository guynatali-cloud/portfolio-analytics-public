# Data

This folder stores project datasets.

## Structure
- `raw` — synthetic surgical operations dataset (1,200 records)
- `cleaned` — transformed dataset with derived KPIs and AI efficiency scoring
- `sample` — smaller shareable subset (150 rows) for portfolio use

## AI Assistance Used
AI was used here as a support tool for planning, documentation, code drafting, KPI definitions, and workflow suggestions.
Final validation, review, and publishing decisions should still be completed by the analyst.
