# AI Research

## Research Question
Can an AI-powered surgical operations dashboard help physicians and surgeons reduce workload, identify bottlenecks, and improve operating room efficiency?

## Purpose
This folder stores prompts, research framing, AI-use disclosure, and the evaluation logic used in the project.

## Files Included
- `research_framework.md`
- `prompts_used.md`
- `ai_assistance_disclosure.md`

## AI Assistance Used
AI was used here as a support tool for planning, documentation, code drafting, KPI definitions, and workflow suggestions.
Final validation, review, and publishing decisions should still be completed by the analyst.
