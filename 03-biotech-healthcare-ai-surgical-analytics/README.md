# Biotech Healthcare AI Surgical Analytics

## Industry
Healthcare / Biotech

## Research Question
Can an AI-powered surgical operations dashboard help physicians and surgeons reduce workload, identify bottlenecks, and improve operating room efficiency?

## Project Goal
Analyze surgical operations data across 8 departments and 15 surgeons to evaluate whether AI-assisted dashboards reduce delays, lower complication rates, and improve patient satisfaction compared to standard workflows.

## Dataset
**Source:** Synthetic surgical operations data (1,200 records, 2023–2025)
**Generated using:** Python (numpy, pandas) with realistic clinical distributions

## Tools Used
- Excel — Department KPIs, AI vs Standard comparison, surgeon scorecard
- SQL — Schema and 7 analytical queries (GROUP BY, JOINs, window functions)
- Python — Data cleaning, EDA, AI efficiency scoring model
- Tableau — Surgical operations dashboard plan
- AI-assisted analysis — Research framing, KPI logic, code drafting

## Repository Structure
- `data/` — raw, cleaned, and sample datasets
- `excel/` — workbook with 4 analysis sheets
- `sql/` — schema and analytical queries
- `python/` — scripts for cleaning, EDA, and AI-assisted scoring
- `dashboard/` — Tableau dashboard plan and monthly extract
- `ai-research/` — prompts, research framing, and AI-use disclosure
- `outputs/` — summary tables, executive summary, recommendations
- `screenshots/` — charts and process visuals for GitHub
- `notes/` — workflow and upload instructions

## Key Questions Answered
1. Does AI-assisted dashboard usage reduce average surgery delays?
2. Which departments have the highest complication and readmission rates?
3. How does patient satisfaction differ between AI and standard workflows?
4. Which surgeons and departments should be prioritized for AI expansion?
5. What is the operating room utilization across different rooms and shifts?

## Key Findings
- AI-Assisted surgeries show lower average delays than Standard workflows
- Complication and readmission rates are reduced with AI dashboard support
- Patient satisfaction scores are higher in AI-Assisted cases
- Neurology and Cardiology departments show highest optimization potential
- AI efficiency scoring identifies priority departments for dashboard expansion
