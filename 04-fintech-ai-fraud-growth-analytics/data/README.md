# Data
This folder stores project datasets.
## Structure
- `raw` - original Kaggle files (10 CSVs)
- `cleaned` - merged dataset with AI risk scoring
- `sample` - smaller shareable subset for portfolio use
## AI Assistance Used
AI was used here as a support tool for planning, documentation, code drafting, KPI definitions, and workflow suggestions. Final validation, review, and publishing decisions should still be completed by the analyst.
